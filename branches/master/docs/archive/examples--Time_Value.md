# Examples of `Time Value`s

Examples of the `time_unit` property of a `Time Value`:

* nanosecond (1/1000000000)

* video frame period (1/25, 1001/30000)

* audio sample period (1/48000)

* Facebook 'Flick' (1/705600000)
